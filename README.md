# ChargerControl

ChargerControl is an Android utility designed for rooted devices to manage battery longevity by controlling charging limits and enabling bypass mode.

## Features
- **Charging Limit**: Automatically stop charging at a specific percentage.
- **Bypass Mode**: Power the device directly from the charger without stressing the battery.
- **Service Status**: Persistent background monitoring using Foreground Services.
- **Material 3 UI**: Modern and clean interface built with Jetpack Compose.
